# 查看 Go 的環境變數
```
go env
```
# 以將 GOPATH 設定為 $HOME/go_vue_chat 為例
```
export GOPATH=/home/vhost/go_vue_chat
```

# 安裝 go iris
```
go get -u github.com/kataras/iris
```

# 安裝 mongodriver
```
go get -u go.mongodb.org/mongo-driver
```

# 安裝 Go-MySQL-Driver
```
go get -u github.com/go-sql-driver/mysql
```

# 更改 /etc/hosts
```
127.0.0.1 chat.go
```

# start
```
go run main.go
```
